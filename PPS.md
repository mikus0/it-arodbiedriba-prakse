Šī nelielā web lietotne parāda datu glabāšanu Supabase datubāzē un to rādīšanu statiskā tīmekļa lapā. Mērķis ir parādīt, kā praktikants var izveidot pilnu piegādājamu piemēru: datu avots (Supabase), kods (HTML/JS) un dokumentācija (PPS, Lietotāja rokasgrāmata, tehniskās piezīmes).

- Praktikants (izstrādātājs), kurš veic uzdevumu.
- IT arodbiedrības biedrs, kurš apskata sarakstu.
- Pasniedzējs vai vērtētājs, kurš pārbauda, vai projekts strādā.

Lietotājs var atvērt publisko GitHub Pages lapu.
- Redz sarakstu ar `members` tabulas ierakstiem (id, name, description)

Nākotnē var pievienot:
- Ierakstu pievienošanu (CREATE).
- Ierakstu rediģēšanu (UPDATE).
- Ierakstu dzēšanu (DELETE).
